# Slides colocados manualmente

Podes copiar PDFs para esta pasta e referenciá-los em `data/academic-data.json`.

Exemplo de material:

```json
{
  "id": "material_poo_tp08_2024",
  "semesterId": "sem_2025_26_2",
  "courseId": "course_poo",
  "lessonId": "lesson_poo_tp08",
  "title": "Slides · Herança e polimorfismo",
  "academicYear": "2024/25",
  "kind": "slides",
  "source": "url",
  "url": "assets/slides/poo-tp08-2024-25.pdf",
  "fileName": "poo-tp08-2024-25.pdf"
}
```

Usa nomes sem espaços quando possível. Um material com o mesmo `academicYear` do semestre ativo aparece sem etiqueta; anos anteriores aparecem sempre identificados.
